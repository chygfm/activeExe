## Welcome to py2AcvtiveEXE  python 2.7 win32  for Windows
You have 2 ways to generate a standalone python program .
Method 1 call py2AcvtiveEXE component from other languages
Method 2 config the /py/py.cfg, Limited functionality,No input interface

Use python's py2AcvtiveEXE component (<15M) to generate a standalone program for C,python,C#,C++,javascript,Go,Java,vbs...languages' calls.
 py2AcvtiveEXE component is too small,only about 15M,too easy,can call almost python program

###Method 1
:: install py2ActiveEXE Component one time for initialization in new machine on windows system.
:: py2ActiveEXE.exe /r  
:: or  click py2ActiveEXE.exe

:: run demos under text folder

:: Uninstall py2ActiveEXE Component
py2ActiveEXE.exe /u

###Method 2
config /py/py.cfg ,change run=1，click py2ActiveEXE.exe,OK!

有2种方法可以生成python的独立运行程序。
方法1 使用其它语言调用py2AcvtiveEXE组件
方法2 修改/py/py.cfg配置文件，能实现简单功能，没有输入界面

使用python的py2AcvtiveEXE组件(<15M)可以生成独立运行程序,以便C,python,C#,C++,javascript,Go,Java,vbs...等语言可以调用。

### 方法1调用组件
:: 在新的电脑上windows系统上需要初始化py2ActiveEXE组件
:: py2ActiveEXE.exe /r  
:: 或者点击运行 py2ActiveEXE.exe

:: 运行test文件夹下面的demo

:: 卸载py2ActiveEXE组件
py2ActiveEXE.exe /u

### 方法2修改/py/py.cfg
修改/py/py.cfg配置文件，run=1，点击运行py2ActiveEXE.exe
